# xdsa

A collection of data structures and algorithms implemented in C for **_educational purposes only_**. It's written in standard C (C99) and uses GNU Make for building.

## Data Structures

- [x] Dynamic Array
- [x] Singly Linked List
- [ ] Doubly Linked List
- [ ] Stack
- [ ] Queue
- [ ] Heap
- [ ] Binary Search Tree
- [ ] AVL Tree

## Algorithms

- [x] Binary Search
- [ ] Insertion Sort

## Usage

### Installation

Clone the repository.

```bash
git clone git@github.com:ragibasif/xdsa.git
```

Make sure you have `gcc` and `make` installed.

### Build the project

```bash
make
```

### Run the executable

```bash
./xdsa
```

### Clean build files

```bash
make clean
```
